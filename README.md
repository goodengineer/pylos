# pylos

Web version of [pylos](https://en.wikipedia.org/wiki/Pylos_(board_game) board game

## requirements
* node version: 12.14+
* yarn version: 1.21+

## development setup

```sh
# install dependencies
yarn install

# automatically build project on changes
yarn watch
```

in a separate terminal run an http server from `dist` directory

## run tests

```sh
# linux/mac
yarn test

# windows
yarn win:test
```
